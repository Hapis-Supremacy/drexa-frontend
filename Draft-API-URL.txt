Drexa API URL Draft

Base URL:
http://localhost:8080/api/v1

Frontend helper:
Use "@/lib/api" with paths that start after /api/v1.
Example:
api.post("/auth/signin", body)
=> http://localhost:8080/api/v1/auth/signin
